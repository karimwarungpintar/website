# nLazy

### Usage
Insert into your html
```html
<script src="nLazy.js"></script>
```
Set your divs
```html
<div class="nl-image" data-background="my-great-image.png">
  <p>My great content</p>
</div>
```

### Statuses
|status|class|
|---|---|
|Not triggered|`nl-image`|
|Loading image|`nl-loading`|
|Image loaded|`nl-loaded`|
